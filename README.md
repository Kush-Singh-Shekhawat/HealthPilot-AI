# Installation

Clone the repository

```powershell
git clone <repo-url>
```

Create virtual environment

```powershell
python -m venv .venv
```

Activate

```powershell
.venv\Scripts\Activate.ps1
```

Install dependencies

```powershell
pip install -r requirements.txt
```

Create `.env`

```env
GOOGLE_API_KEY=YOUR_API_KEY
```

Run

```powershell
streamlit run app.py
```
